# 素履 - 推广文案

## 酷安发布文案

### 标题
素履 - 一款真正保护隐私的本地日记 App

### 正文
厌倦了日记 App 要注册账号、要联网、担心隐私泄露？

素履，一款以「绝对隐私」为核心的日记应用。

**核心特点：**
- 所有数据本地存储，不上传任何服务器
- PIN 码 + 生物识别双重保护
- 支持 Markdown 写作，实时预览
- 每篇日记可添加 9 张图片
- 心情/天气记录，7 天趋势统计
- 5 种主题配色，深色模式
- ZIP 格式导出，数据完全属于你

**为什么选择素履？**
市面上大多数日记 App 都需要注册、联网，你的私密文字可能被存储在某个服务器上。素履不一样 —— 没有账号系统，没有云同步，你的日记只存在于你的手机里。

名字取自《周易》"素履之往，独行愿也"，寓意素雅纯粹，记录人生轨迹。

开源地址：github.com/pilipili-hash/trace-diary
