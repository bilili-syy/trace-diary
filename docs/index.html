核心特性

绝对隐私

所有数据存储在本地设备，不上传任何服务器。支持 PIN 码和生物识别双重保护。

Markdown 写作

支持完整的 Markdown 语法，让写作更加专注。实时预览，撤销重做，随心所欲。

图片日记

每篇日记支持添加最多 9 张图片，自动压缩优化，完整记录生活的每个瞬间。

心情追踪

记录每天的心情和天气，7 天心情趋势图表，了解自己的情绪变化。

日历视图

月历形式浏览所有日记，一目了然查看写作记录，培养持续写作习惯。

数据导出

支持 ZIP 格式完整导出导入，包含所有日记和图片，数据永远属于你。

🔒 你的隐私，我们的承诺

素履不会收集任何用户数据，不需要注册账号，不联网上传。你的日记只存在于你的设备中，真正做到"我的日记我做主"。
